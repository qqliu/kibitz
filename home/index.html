Option 1 creates the recommender for you. For this option, your table must contain the following columns: title (primary key), description, image.

Recommender Types

User-based recommends based on individual user ratings.

Item-based computes item similarity.

You must have a column in your table providing numerical ratings for each item to use overall ratings.

You must choose at least one.

Item-Based/Rating Info

To use overall ratings and item-based, provide the following info.

For item-based, provide column names for at least one comparison column. Ranked by importance.

Rating Info

To use overall ratings, provide the column name containing numerical ratings for each item.

Item-Based Info

To use item-based, provide column names for at least one comparison columnn used to compute similarity. Ranked by importance.
